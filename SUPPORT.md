# Support

For bug reports and feature requests, please use GitHub Issues first:

https://github.com/lx544690189/SQLite-Tool/issues

When filing an issue, please include:

- VS Code version and operating system.
- sqlite-tool version.
- Steps to reproduce the problem.
- Whether it only happens with a specific database file.
- Relevant screenshots or error messages.

Please keep a backup before editing important databases.
