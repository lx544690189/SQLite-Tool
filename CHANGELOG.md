# 更新日志

## 2.0.0

本版本基于全新架构重构（含权限模型重构），相较 v1.x 为重大升级。

- 支持以自定义编辑器打开 `.db`、`.sqlite`、`.sqlite3` 文件。
- 支持浏览表、分页查看数据、排序、全字段搜索和指定字段搜索。
- 支持新增、编辑、删除数据行，并处理 NULL、主键保护和删除确认。
- 支持查看建表 SQL、重命名表、通过表单创建新表。
- 支持在 Monaco SQL 编辑器中执行查询和写入语句，并查看执行历史。
- 支持跟随 VS Code 明暗主题，以及工具内主题切换。
- 支持通过 VS Code 原生脏标记和 `Ctrl/Cmd+S` 保存流程写回数据库文件。
- 支持中文、英文、法语、日语、韩语界面文案。
