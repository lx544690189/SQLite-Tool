# sqlite-tool

[中文](README.zh-CN.md) | [English](README.md) | [Français](README.fr.md) | 日本語 | [한국어](README.ko.md)

sqlite-tool は VS Code 向けの SQLite データベース可視化ツールです。`.db`、`.sqlite`、`.sqlite3` ファイルを開くだけで、テーブルの閲覧、データの確認と編集、テーブル作成、SQL 実行をエディター内で行えます。

## 機能

- 追加設定なしで SQLite データベースファイルをカスタムエディターで開けます。
- テーブル一覧、行数、ページングされたデータ、ソート結果を確認できます。
- 全フィールドまたは指定フィールドでデータを検索できます。
- NULL の扱い、主キー保護、削除確認付きで行の追加、編集、削除ができます。
- CREATE SQL の表示、テーブル名変更、フォームによる新規テーブル作成に対応します。
- Monaco SQL エディターでクエリや更新系 SQL を実行し、結果と履歴を確認できます。
- VS Code のライト/ダークテーマに追従し、ツール内で手動切り替えもできます。
- VS Code 標準の未保存状態と `Ctrl/Cmd+S` の保存フローで変更をディスクへ書き戻します。

## インストール

`.vsix` パッケージがある場合は、次のコマンドでインストールできます。

```bash
code --install-extension sqlite-tool-0.0.1.vsix
```

VS Code の拡張機能ビューから「Install from VSIX...」を選んでインストールすることもできます。

## 使い方

1. VS Code のエクスプローラーで `.db`、`.sqlite`、`.sqlite3` ファイルを開きます。
2. ファイルは自動的に `sqlite-tool` エディターで開かれます。
3. 左側でテーブルを選択し、データを閲覧、検索、編集します。
4. SQL を実行したい場合は SQL 実行ビューへ切り替えます。
5. 変更後はエディタータブが未保存状態になります。`Ctrl/Cmd+S` でデータベースファイルへ書き込みます。

## 設定

sqlite-tool はインターフェース言語の設定に対応しています。

- `Auto`: VS Code の表示言語に従います。
- `Chinese` / `English` / `French` / `Japanese` / `Korean`: 手動で言語を指定します。

VS Code の設定で `sqlite-tool` を検索するか、ツール内の設定パネルから言語、テーマ、既定のページサイズ、SQL エディターのフォントサイズを変更できます。

## 注意事項

- 重要なデータベースを編集する前にバックアップを残してください。
- 主キーがなく `rowid` も利用できないテーブルは閲覧専用で、行単位の編集や削除はできません。
- ファイルが外部で変更された場合、保存前に警告を表示し、外部変更の上書きを避けられるようにします。
